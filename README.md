# C-Strike
## Requirements:
A Linux distro with compatible sockets API available
## Building:
Run C compiler- for instance:
gcc C-strike.c -o C-Strike
## Usage 
Disclaimer: This tool should be used only for testing the systems response to high load in a controlled environment. Unauthorized use is forbidden and may result in legal repercussions depending on local law.
C-Strike [-T target] [-t type] [-s source] [-p port number] [-n n_processes] [-d duration] [-l packet length]
